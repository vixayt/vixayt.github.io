# Vixay Portfolio

Github Repo: https://github.com/vixayt/vixayt.github.io

Published at: https://vixayt.github.io/

## **What worked:**

What worked in my favor was already knowing some CSS. It was nice not having to look up how to do basic things like `background` or `color`. It was very beneficial because I knew how powerful CSS and Bootstrap were. Using this to my advantage, I had a better idea of what to look up and was able to understand (most) of everything that was done.

## **What didn't work:**

One thing I wish I had more time to do, and will eventually do, is implementing the `@media` to make my website more mobile friendly. The email form doesn't work, so I added a little javascript that would just clear the form.

## **Did I install anything:**

- I am using the Bootstrap library.
- In order to use icons more easily, I am using ther Font Awesome library.

## **Issues I ran into:**

The biggest issues I ran into were all CSS related. The more I learn about CSS, the more I realize I how much CSS I don't know.  
One smaller issue I had was with the email form. The submit button doesn't actually do anything, it just clears the form.

## **How did I solve those issues:**

I relied heavily on the bootstrap documentation to look up the syntax of how to do things in bootstrap.

Another resource I used a lot was w3schools.com.
